•SQLMap (SQL Injection Automated Penetration Testing Tools)

SQLMap adalah sebuah perangkat lunak open-source yang digunakan untuk mengotomatisasi pengujian penetrasi terhadap aplikasi web dengan fokus pada menemukan dan mengeksploitasi kerentanan SQL injection. Ini membantu para profesional keamanan dan peneliti keamanan untuk mengidentifikasi dan memanfaatkan kerentanan SQL injection dalam aplikasi web untuk tujuan pengujian keamanan. Baca selengkapnya
